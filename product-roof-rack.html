Installation Guide

Installation Overview

This racking system is designed for straightforward installation by professionals or experienced DIY installers. The kit includes all components needed for a complete installation.

Installation Steps

  1. Plan Layout: Determine panel placement and orientation on roof
  2. Mark Rafter Locations: Use stud finder to locate roof rafters/trusses
  3. Install Roof Attachments: Secure Flashfoot or L-feet to rafters with flashing
  4. Mount Rails: Attach aluminum rails to roof attachments
  5. Install End Clamps: Place end clamps at rail ends
  6. Mount First Panel: Lift and secure first panel with clamps
  7. Install Mid Clamps: Secure between panels as you add each one
  8. Ground System: Connect bonding jumpers for electrical safety
  9. Final Inspection: Verify all connections and tighten to torque specs

Tools Required

  • Impact driver or drill
  • Socket set (10mm, 13mm)
  • Torque wrench (recommended)
  • Tape measure and chalk line
  • Stud finder
  • Safety equipment (harness, gloves, non-slip shoes)

Safety Notes

⚠️ Important: Roof work is dangerous. Always use proper fall protection equipment. If you're not comfortable working on a roof, please hire a professional installer. Contact us for recommendations in your area.
